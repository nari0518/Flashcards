# フラッシュカード (Flashcards)

自分で作成した問題セットを使って、効率的に漢字の「読み・書き」や「英単語・表現」を学習できる汎用フラッシュカードWebアプリケーションです。
シンプルで直感的な操作性、高い視認性（ダークモード）、そしてデバイスを問わず快適に動作するレスポンシブデザインを特徴としています。

**[実際のアプリを開く](https://nari0518.github.io/Flashcards/)**

## 主な機能

### 1. マルチモード学習
- **漢字モード**: 
    - **読み**: キーボード入力による正誤判定、または表示された答えを確認する「判定のみ」モード。
    - **書き**: 例文を見て紙などに書き、答えを表示して自分で判定。
- **英語モード**: 
    - **英 ➜ 日 / 日 ➜ 英**: 単語を見て意味を思い出し、答えを表示して判定する双方向学習。

### 2. 学習支援と履歴管理
- **履歴の記録**: 学習日時、モード、セット名、スコアを記録。
- **間違いの復習**: 履歴から「どの問題をどう間違えたか」を詳しく確認可能。
- **データの入出力**: 学習履歴をJSON形式でエクスポート・インポートでき、デバイス間でのデータ移行も容易。

### 3. 問題セットの柔軟な管理
- **フォルダ自動読み込み**: `public/sets/` 配下の `Kanji/` または `English/` フォルダにCSVを置くだけで自動登録。
- **UIからの管理**: アプリ上での新規セット作成、名称変更、CSVインポート/エクスポートに対応。

## 独自の問題セットを追加する方法

### CSVファイルの作成

学習したい内容に合わせて、以下の形式でCSVファイルを作成します。

#### 漢字セットの場合
`public/sets/Kanji/` フォルダに配置します。
```csv
漢字,読み,例文
学校,がっこう,明日から（学校）が始まる。
勉強,べんきょう,毎日（勉強）をする習慣をつける。
```
※ 例文内の `（ ）` で囲んだ部分がクイズの対象（表示/非表示の切り替え）になります。

#### 英語セットの場合
`public/sets/English/` フォルダに配置します。
```csv
word,meaning
apple,りんご
experience,経験
```

### 反映方法
1. 作成したCSVを `public/sets/` 内の適切なサブフォルダ（`Kanji` または `English`）に保存します。
2. アプリをリロードすると自動的に新しいセットが読み込まれます。
3. または、アプリの「セット管理」画面からCSVファイルを直接インポートすることも可能です。

## 開発（ローカル起動）

このアプリは **React** と **Vite** で構築されています。

```bash
# インストール
npm install

# 開発用サーバー起動
npm run dev
```

## 使用技術

- **React** (Hooks: useState, useEffect, useRef)
- **Vite** - 高速なビルドツール
- **react-markdown** - README等のドキュメント表示用
- **Vanilla CSS** - カスタムプロパティによるレスポンシブデザイン
- **LocalStorage** - ブラウザでのデータ永続化
- **GitHub Actions** - CI/CD自動デプロイ
